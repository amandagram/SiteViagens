# 🌍 ExploreMundo

Site institucional de turismo desenvolvido em HTML e CSS puro, sem frameworks ou dependências externas.

---

## 📄 Páginas

- **index.html** — Página principal com banner, destinos populares, sobre nós e contato
- **destinos.html** — Página com todos os destinos organizados por categoria

---

## 🗂️ Estrutura do Projeto

```
exploremundo/
│
├── index.html
├── destinos.html
└── README.md
```

---

## 🎨 Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox, variáveis CSS)
- Google Fonts (Playfair Display + Nunito)

---

## 🚀 Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/exploremundo.git
   ```

2. Abra o arquivo `index.html` no navegador — nenhuma instalação necessária.
